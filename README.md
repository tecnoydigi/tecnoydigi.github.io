# Recursos Web de Libros Educativos

Repositorio para la web de recursos complementarios de los libros
publicados por **DVL Ediciones**.

Esta web incluye:
- Navegación por libro y capítulo
- Buscador global de recursos
- Filtros por temática
- Multilenguaje (EN / ES)
- Internamente protegido por aviso de copyright

## Uso / Desarrollo

Clona el repo y abre `index.html` en tu navegador.
